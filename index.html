Blind OCR API

Accurate and Cost-Effective Text Extraction for Everyone

API Documentation

Welcome to the Blind OCR API. This service provides high-quality text extraction from PDF and image files using the Tesseract open-source model.

Endpoints:
  • POST /ocr/pdf: Upload a PDF file to extract text.
  • POST /ocr/image: Upload an image file to extract text.

For more details, visit the Swagger UI.
